# copilot-instruction-kit

GitHub Copilotを使用してエージェントアプリケーション（Agentic Applications）を開発する際の、効率的な指示書（Custom Instructions）テンプレート集です。

## 📦 セットアップ

このリポジトリに含まれるテンプレートを使用するには、以下の手順でファイルを配置してください。

1. テンプレートファイルを選択し、ファイル名を **`copilot-instructions.md`** に変更します。
2. プロジェクトのルートディレクトリにある **`.github`** フォルダ内に配置します。

配置例:
```text
.github/
└── copilot-instructions.md
